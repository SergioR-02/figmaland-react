# FigmaLand React — mini plantilla

Demo: https://figmaland-react.vercel.app


-----------------
Pequeña aplicación React + TypeScript creada con Vite. Contiene componentes simples (Navbar, Hero, Features, Contact, Footer) y usa CSS clásico para estilos.

Tecnologías
-----------------
- React (TSX)
- TypeScript
- Vite (dev server + build)
- CSS (archivos )
- ESLint
- Node.js & npm

Estructura relevante
-----------------
- `src/components/` — componentes de UI (cada componente suele tener su CSS junto al TSX)
- `src/assets/` — imágenes y vídeos utilizados como módulos importados
- `public/` — (opcional) archivos estáticos que se copian tal cual al build

Comandos rápidos (PowerShell)
-----------------
```powershell
npm install
npm run dev     # modo desarrollo (Vite)
npm run build   # build de producción
npm run preview # sirve la build localmente (simula producción)
```
